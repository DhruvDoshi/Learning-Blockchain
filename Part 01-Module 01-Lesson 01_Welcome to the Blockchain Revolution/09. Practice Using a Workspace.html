# Practice Using a Workspace

In the previous concept, you were introduced to workspaces. Now's your chance to practice playing in your very first workspace!

Let's Practice Using Your Workspace following the Steps below:

Step 1 Open up a Terminal and check your Node's version ( node -v )
Step 2 Create a file newFile.js ( touch newFile.js )
Step 3 Open up newFile.js from the Navigator Section
Step 4 In newFile.js, write console.log ("Yayy..I am enjoying my Blockchain Nanodegree!!")
Step 5 Run this script file in your terminal to see your message ( node newFile.js )
