10. Explore Bitcoin's Codebase

Bitcoin's Codebase

Now that we've seen the data that exists in a blockchain, it's helpful to understand how these ideas are built and managed. Luckily, many blockchains are open source, and in particular Bitcoin has an enormous open source codebase that we can use to study and understand these blockchain concepts.

Explore Bitcoin's Codebase

In the next video, we'll take a look at this codebase and go over a few key points you can use to help understand how data handled in the Bitcoin blockchain.

Wrap Up

Going through the codebase of any open source repo is a great way to understand how the software works. Keep in mind, these can be very large repositories and it could be easy to get lost at first. What helps is to focus on a few key questions you have, then narrow in on very specific parts of the code that can help answer your question.

Next, we'll go over something you may have thought of…who makes updates to this code? It's an interesting idea, especially if you're unfamiliar with open source projects. The answer has to do with Bitcoin Improvement Proposals, and we'll cover these in more detail in the next section.
