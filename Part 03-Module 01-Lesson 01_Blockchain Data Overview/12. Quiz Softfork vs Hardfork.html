12. Quiz: Softfork vs Hardfork

Hardfork, Softfork, or Software fork?

In the last section, we went over Bitcoin Improvement Proposals, and how they are related to different types of forks. In this section, try using this information to help you through the following problem set.

In the examples below, decide whether each is a softfork, hardfork, or software fork.

Scenario 1

Blocks Violating the New Rule are made stale by the upgraded mining majority
SOLUTION: Hardfork

Scenario 2

Non-upgraded Node Rejects the new Rule resulting in a Diverging Chain.
SOLUTION: Softfork

Scenario 3

Which type of forking doesnâ€™t require any nodes to upgrade to maintain consensus since all blocks with the new forked in rules also follow the old rules, therefore old clients accept them. Also, this type of forking cannot be reversed.

SOLUTION: Softfork

Scenario 4

Litecoin is considered to be which type of forking?
SOLUTION: Software fork
