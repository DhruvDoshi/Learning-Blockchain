Problem Set: P2P or Client-Server?

Coming up, we put together a variety of scenarios. In each of these use cases, you'll decide whether a P2P model or a client-server model would be more appropriate. To help decide, think about each of the topics we went over previously and try to come up with a solution that is most relevant to the situation.

SOLUTION: Client-Server Model

SOLUTION: P2P Network

SOLUTION: P2P Network

SOLUTION: Client-Server Model

Wrap Up
