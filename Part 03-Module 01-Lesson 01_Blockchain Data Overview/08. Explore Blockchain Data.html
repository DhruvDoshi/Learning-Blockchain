Problem Set: Blockchain Data

Throughout the past couple videos, we went over some new terminology and saw how it is represented in the blockchain dataset. To help make sure you're prepared for future concepts, we'd like you to think through some of the questions below.

Blockchain Data

When reviewing Blockchain data, what does the block height mean within the header of a block?
SOLUTION: Block number in the chain

Blockchain Data

When reviewing Blockchain header data, which of the following fields contribute to block immutability?
SOLUTION:
  • Previous Hash
  • Nonce
  • Bits
  • Time Stamp

Wrap Up

In this section, we went beyond the blockchain as a dataset and into the actual blockchain data. First, we went through a demonstration using an online block explorer, then followed this up by looking at the same data in JSON.

For now, focus more specifically on how the data you're seeing relates to the concepts you learned previously. This will help tie together the knowledge you have already and allow you solve more complex problems later on.
