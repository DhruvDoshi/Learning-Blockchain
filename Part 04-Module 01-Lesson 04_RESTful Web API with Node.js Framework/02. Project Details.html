# Build Your Project

For this project, you’ll build a RESTful API using a Node.js framework that will interface with the private blockchain you completed Project 2 - "Private Blockchain".
You’ll want to build this project out in steps.

Step 1 Select a Node.js framework
Step 2 Configure API service to the appropriate port
Step 3 Configure two endpoints - GET & POST
Step 4 Update project README.md
