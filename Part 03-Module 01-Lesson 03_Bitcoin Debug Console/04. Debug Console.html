04. Debug Console

Debug Console

In this section, we'll go over the bitcoin core debug console. This is a powerful tool that helps you explore data within the bitcoin blockchain. We'll be using this debug console throughout the rest of the lesson to learn many of the commands you can use to access data for yourself.

Wrap Up

Coming up, we'll explore many of the commands available to you in the Bitcoin Core Debug console. The goal of these sections is to help you feel confident working with this data and exploring commands for yourself.

They're covered in a specific order to make sure you've seen certain commands that you may need before moving on to others. That said, they're not too dependent on each other. If you are interested in a certain section, feel free to jump straight to that, and if you're confident going off to explore on your own, that's great too!
