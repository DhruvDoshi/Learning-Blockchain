03. Terminal Overview

Get Started with Bitcoin Core using the Debug Console

Coming up, we'll explore the Bitcoin Blockchain using the Debug Console.

At this point you have hopefully made the decision about whether or not you are choosing to download the network to your local computer, or if you'd prefer to work in our online terminal.

To quickly recap the guidelines on how to best make this decision for yourself, check out the concept from the previous lesson titled Data Warnings and Bitcoin.config.

Use the Bitcoin Core Debug Console

If you are using this on your local computer then you're all set up and ready to go! In the next concept, we'll go over the basics of the Bitcoin Core Debug Console. Throughout the rest of the lesson we'll review commands that help us investigate the Bitcoin Blockchain.

Use the Classroom Workspace

If you are using the classroom workspace then read through the section below to get started.

Get Started with the Classroom Workspace

Throughout this lesson we'll continue using the debug console on our local computer. The workspace is meant to help support you in the case that you do not have this software installed.

Using this workspace, you have access to all of the commands available in the debug console, and can experiment with whatever commands you'd like.

For this lesson, you can find the classroom workspace here. Bitcoin CLI Workspace >

You can also find this in the concept near the end of the lesson labeled Bitcoin-CLI Workspace

Using the Workspace

When first getting started, you'll see a blank workspace.

  • To get started click NEW TERMINAL on the bottom workspace
  • Type bitcoin-cli help into the terminal window

Coming up, we'll go over what bitcoin-cli is useful for. For now you can use this to test that this workspace is working correctly.

If you're workspace looks like the image below, then you're all set up and ready to go!

The Difference is bitcoin-cli

One difference when using the workspace you'll need to start each command with bitcoin-cli. In our tutorials, we do not use this since we are in the debug console but keep this in mind as you progress through the lesson.

Wrap Up

Hopefully this helps you as you get started with your first commands. Throughout the rest of the lesson, we'll be going over many commands that can help you get started with these new tools.
