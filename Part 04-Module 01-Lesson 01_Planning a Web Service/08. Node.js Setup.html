08. Node.js Setup

Node.js Setup

Step 1

Run this command to see if you already have node on your computer and if so, what version. Windows (Command prompt), Mac/Linux (Terminal).

  • node --version

Step 2

If either it shows you do not have node installed yet, or do not have the latest stable version, navigate to nodejs.org and install the latest version.

At the time of this release, 8.11.3 LTS is the latest stable version which I'd recommend.

Step 3

Re-run the node --version command to verify the correct version of Node.js has installed. Windows (Command prompt), Mac/Linux (Terminal).

Step 4

We are going to create our first Node.js application:

  • Create a new folder and name it my-nodejs-project using the command: mkdir my-nodejs-project

  • Move to the folder you just created cd my-nodejs-project

  • Now that you are in the folder my-nodejs-project we are going to initialize our Node.js project with the command npm init

Vm-Create-Project

  • Let's create our index.js file:

  • Write the following code in your index.js file:
console.log("Using Node.js for my Blockchain Program");

You can open the file with any text editor like Notepad (Windows), TextEdit (MAC OS), or any specialized code editor like VSCode, Sublime or ATOM.

  • Now let's test our application using the command node index.js
